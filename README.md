# Gold Captação

Sistema simples para comparar imóveis captados no SP Imóvel com os imóveis cadastrados na Gold Leste.

## Rodar local

```bash
pip install -r requirements.txt
python main.py
```

Abra: http://localhost:5000

## Deploy grátis no Replit

1. Crie um Repl Python.
2. Envie todos os arquivos.
3. No botão Run, use: `python main.py`.

## Deploy Render/Koyeb/Railway

Build command:
```bash
pip install -r requirements.txt
```

Start command:
```bash
python main.py
```
